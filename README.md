Análisis Exploratorio y Limpieza de Datos de Aeropuertos en Argentina: Calidad, Distribución y Conectividad
Este proyecto analiza la calidad y estructura de los datos de aeropuertos en Argentina, con el objetivo de transformar datos crudos en información confiable para la toma de decisiones.

Se aplicaron técnicas de limpieza, normalización y transformación de datos utilizando Python, seguido de un análisis exploratorio y visualización en Power BI para identificar patrones geográficos, operativos y de conectividad.

El enfoque del proyecto simula un entorno profesional donde los datos presentan inconsistencias, valores nulos y problemas de formato.
